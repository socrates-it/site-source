## How to contribute ?

The site is build with [Hugo](https://gohugo.io/) and the content is in [config.toml](config.toml).

### Some commands

Run the site locally:

```shell
hugo server
```

Build the site locally:

```shell
hugo
```

Use
```shell
./publish.sh
```
to copy the content to `../socrates-it.github.io/` for publishing.
